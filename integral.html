Lũy thừa
x^n → xⁿ (VD: x^2, x^3, x^(1/2))
Căn bậc hai
sqrt(x) → √x
Hàm mũ
exp(x) → eˣ  |  e^x → eˣ
Logarit tự nhiên
log(x) → ln(x)
Logarit cơ số 10
log10(x) → log₁₀(x)
Lượng giác
sin(x), cos(x), tan(x)
Hàm trị tuyệt đối
abs(x) → |x|
Hằng số
pi → π  |  e → e
Ví dụ mẫu
x^2 + 2*x + 1
sin(x)*cos(x)
exp(-x^2)
1/(1+x^2)
sqrt(1-x^2)
Hàm hằng
\((c)' = 0\)
Lũy thừa
\((x^n)' = n x^{n-1}\)
Căn bậc hai
\((\sqrt{x})' = \dfrac{1}{2\sqrt{x}}\)
Hàm mũ tự nhiên
\((e^x)' = e^x\)
Hàm mũ tổng quát
\((a^x)' = a^x \ln a\)
Logarit tự nhiên
\((\ln x)' = \dfrac{1}{x}\)
Logarit cơ số a
\((\log_a x)' = \dfrac{1}{x \ln a}\)
Sin
\((\sin x)' = \cos x\)
Cos
\((\cos x)' = -\sin x\)
Tan
\((\tan x)' = \dfrac{1}{\cos^2 x}\)
Cot
\((\cot x)' = -\dfrac{1}{\sin^2 x}\)
Tổng / Hiệu
\((u \pm v)' = u' \pm v'\)
Tích
\((uv)' = u'v + uv'\)
Thương
\(\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}\)
Hàm hợp (Chain rule)
\([f(g(x))]' = f'(g(x)) \cdot g'(x)\)
VD: \((e^{u})' = e^u \cdot u'\)
\((\sin u)' = \cos u \cdot u'\)
\((\ln u)' = \dfrac{u'}{u}\)
Hằng số
\(\int k\, dx = kx + C\)
Lũy thừa \((n \neq -1)\)
\(\int x^n dx = \dfrac{x^{n+1}}{n+1} + C\)
Nghịch đảo
\(\int \dfrac{1}{x} dx = \ln|x| + C\)
Căn bậc hai
\(\int \sqrt{x}\, dx = \dfrac{2}{3}x^{3/2} + C\)
Mũ tự nhiên
\(\int e^x dx = e^x + C\)
Mũ cơ số a
\(\int a^x dx = \dfrac{a^x}{\ln a} + C\)
Sin
\(\int \sin x\, dx = -\cos x + C\)
Cos
\(\int \cos x\, dx = \sin x + C\)
Tan
\(\int \tan x\, dx = -\ln|\cos x| + C\)
Cot
\(\int \cot x\, dx = \ln|\sin x| + C\)
\(\frac{1}{1+x^2}\)
\(\int \dfrac{1}{1+x^2} dx = \arctan x + C\)
\(\frac{1}{\sqrt{1-x^2}}\)
\(\int \dfrac{1}{\sqrt{1-x^2}} dx = \arcsin x + C\)
Tích phân từng phần
\(\int u\, dv = uv - \int v\, du\)
Chọn u theo LIATE: Logarit → Nghịch giác → Đại số → Lượng giác → Mũ
Đổi biến (t = g(x))
\(\int f(g(x))g'(x)\,dx = \int f(t)\,dt\)
Newton–Leibniz
\(\int_a^b f(x)\,dx = F(b) - F(a)\)
F(x) là nguyên hàm của f(x)
Tích phân từng phần (xác định)
\(\int_a^b u\,dv = \bigl[uv\bigr]_a^b - \int_a^b v\,du\)
Diện tích hình phẳng (f ≥ g)
\(S = \int_a^b [f(x)-g(x)]\, dx\)
Nếu không rõ dấu: \(S = \int_a^b |f(x)-g(x)|\, dx\)
Diện tích quanh trục Ox
\(S = \int_a^b |f(x)|\, dx\)
Thể tích tròn xoay quanh Ox
\(V = \pi \int_a^b [f(x)]^2\, dx\)
Thể tích tròn xoay quanh Oy
\(V = \pi \int_{c}^{d} [x(y)]^2\, dy\)
Đổi biến: viết x theo y, đổi cận sang y
Thể tích giữa 2 hàm quanh Ox
\(V = \pi \int_a^b \bigl([f(x)]^2 - [g(x)]^2\bigr)\, dx\)
f(x) ≥ g(x) ≥ 0
Thể tích vật thể (biết S(x))
\(V = \int_a^b S(x)\, dx\)
S(x): diện tích mặt cắt vuông góc trục Ox
Độ dài đường cong
\(L = \int_a^b \sqrt{1 + [f'(x)]^2}\, dx\)
